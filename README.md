# Working with Remote Repositories

This repository contains practice files and exercises for learning how to work with **remote repositories** on GitHub.

### Contents
- Sample text files (`first.txt`, `firstNew.txt`)
- Excel file (`myexcel.xlsx`)
- Access database (`db.accdb`)
- Practice folders from Day 12 & Day 13
- `.gitignore` file

### Purpose
Created for learning Git basics, especially:
- Connecting local repositories to GitHub
- Pushing and pulling changes
- Working with remote repositories
